# Restaurant Rating Prediction using Machine Learning

This project predicts the aggregate rating of restaurants using machine learning techniques.

## Models Used
- Linear Regression
- Decision Tree Regressor

## Key Features
- Votes
- Price range
- Average cost for two
- Location
- Online delivery & table booking

## Dataset
Zomato-style restaurant dataset containing 9,500+ entries.

## Results
- Decision Tree achieved R² ≈ 0.95
- Linear Regression achieved R² ≈ 0.32
- Votes identified as the most influential feature

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
